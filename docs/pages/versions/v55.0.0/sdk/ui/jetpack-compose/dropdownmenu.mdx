Expo UI DropdownMenu matches the official Jetpack Compose [Menu API](https://developer.android.com/develop/ui/compose/components/menu) and displays a dropdown menu when a trigger element is pressed.

## Usage

### Basic dropdown menu

```tsx BasicDropdownMenuExample.tsx
import { Host, DropdownMenu, Button } from '@expo/ui/jetpack-compose';

export default function BasicDropdownMenuExample() {
  return (
    <Host matchContents>
      <DropdownMenu style={{ width: 150, height: 50 }}>
        <DropdownMenu.Items>
          <Button onPress={() => console.log('Pressed Hello')}>Hello</Button>
          <Button variant="bordered" color="#ff0000" onPress={() => console.log('Pressed Love it')}>
            Love it
          </Button>
        </DropdownMenu.Items>
        <DropdownMenu.Trigger>
          <Button variant="bordered" onPress={() => {}}>
            Show Menu
          </Button>
        </DropdownMenu.Trigger>
      </DropdownMenu>
    </Host>
  );
}
```
